## 프로젝트 소개
식사를 준비할 때 자주 과도한 음식 조리로 인해 음식이 남은 경험이 있나요? <br/>
남은 음식은 전부 쓰레기가 되어서 생활환경을 오염시키고 기후 위기를 초래합니다. <br/>
Ecoportion 이러한 문제를 해결하기 위해 개인의 식사량을 정확하게 예측하고 이에 맞는 맞춤형 레시피를 제안합니다. <br/><br/>
